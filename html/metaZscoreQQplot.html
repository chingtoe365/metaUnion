metaZscoreQQplot {metaUnion}R Documentation

Q-Q plot of meta-analysis z score

Description

Gereate a Q-Q plot figure of meta-analysis z scores

Usage

	metaZscoreQQplot(metaResult, mainTitle, savefile, width=500, height=500)

Arguments

metaResult

A meta-analysis result list generated by function metaAnalysis()

mainTitle

main title of the Q-Q plot figure

savefile

file name of the file

width

width of the figure, default 500

height

height of the figure, default 500

Value

A Q-Q plot figure of meta-analysis z scores

Examples

	metaZscoreQQplot(metaResult, "Q-Q Plot of Z score", "qqplot.jpg")

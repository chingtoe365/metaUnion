metaUnion-package {metaUnion}R Documentation

Meta-analysis for unified Microarrays

Description

Meta-analysis using this package will include in the calculation all genes in the input studies. It does not require the same gene dimension for every study and will take into account the cases where genes are missing from some datasets. It also gives additional useful information in the output matrix including Bonferroni corrected p-value and effect direction.
